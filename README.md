# 技術経営専門職学位課程 修了要件チェッカー

単位確認用アプリです。2026年度学修案内の技術経営専門職学位課程の修了要件を、Excelまたは教務Webシステムの成績一覧PDFから確認します。

## 使い方

1. `run_app.ps1` をPowerShellで実行します。
2. ブラウザでローカルURLが開きます。通常は `http://127.0.0.1:8765/` です。
3. チェックしたい `.xlsx`、`.xlsm`、または `.pdf` を選択して「判定」を押します。

PDF入力は、教務Webシステムの「成績閲覧：成績一覧（大学院）」PDFと同系統の形式を対象にしています。PDFから科目を抽出できない場合は、Excel形式の入力を利用してください。

## exe版

`dist/tim-requirements-checker-v3.exe` を実行すると、Pythonを別途インストールしていないPCでも利用できます。
起動後、表示されたURLをブラウザで開いてください。通常は自動でブラウザが開きます。

## 判定する要件

- 総単位 40単位以上
- 文系教養科目 400番台 2単位以上
- 文系教養科目 500番台 1単位以上
- アントレプレナーシップ科目 GA0M / GA1M
- 講究科目 8単位
- A群400番台、B群、C群、D群またはE群
- 専門科目群（標準学修課程）25単位以上

プロジェクトレポートの審査および最終試験の合否は、修得単位データからは判定できないため注記のみ表示します。
